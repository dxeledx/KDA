# KSDA D.1 / D.1+ 正式结果备忘录

**日期**: 2026-03-08  
**对应运行**:
- `results/ksda/exp_d1/2026-03-08-ksda-d1`
- `results/ksda/exp_d1_plus/2026-03-08-ksda-d1plus`

## 一句话结论

本轮 `KSDA Phase 1` **没有通过受控验证**：
- `D.1` 的 `KSDA-static` 明显低于 `RA`，未达到止损门槛；
- `D.1+` 中 `KCAR gate` 没有优于 `geometric gate`，也没有接近 `RA`；
- 但结果同时表明：**Koopman 表征本身有信息，当前真正失败的是静态对齐器 `A`，以及由此带来的 D.1 / D.1+ 高耦合问题。**

---

## 1. 本轮实验要回答什么

### D.1

回答：

> 在 Koopman 线性空间中，静态 source→target 仿射对齐是否至少不输 `RA`？

### D.1+

回答：

> 在固定 `A / K_s / K_t` 条件下，`KCAR` 是否比简单几何信号更适合作为动态 gate 信号？

本轮的止损规则也很明确：
- 若 `KSDA-static < RA - 1%`，则原则上不应继续 `D.1+`
- 若 `KCAR gate` 只和 `geometric gate` 持平，则不继续 `D.2 / D.3`

---

## 2. D.1 结果：静态对齐没有通过 feasibility gate

### 总表

| 方法 | Mean Acc | Std | Δ vs RA |
|------|----------|-----|---------|
| `RA + CSP + LDA` | `0.4375` | `0.1408` | `0.0000` |
| `Koopman-noalign + LDA` | `0.4109` | `0.1453` | `-0.0266` |
| `NoAlign + CSP + LDA` | `0.3804` | `0.1179` | `-0.0571` |
| `KSDA-static` | `0.3507` | `0.0618` | `-0.0868` |

数据来源：`results/ksda/exp_d1/2026-03-08-ksda-d1/summary.json`

### 关键观察

1. **`Koopman-noalign` 并不差**
   - 它达到 `0.4109`
   - 相比 `RA` 只低 `-2.66%`
   - 说明 **Koopman 表征本身不是完全无效的**

2. **`KSDA-static` 显著更差**
   - `0.3507`
   - 比 `RA` 低 `-8.68%`
   - 比 `Koopman-noalign` 还低 `-6.02%`

3. **D.1 的失败不是“Koopman space 不可用”**
   - 更准确的说法是：
   - **当前版本的静态仿射对齐器 `A` 在 Koopman 特征空间中是错误的、过强的，或不稳定的**

### 结论

`D.1` **未通过**我们事先定义的最低门槛：

> `KSDA-static >= RA - 0.5%`

它不仅没接近这个门槛，而且已经触发了更强的停止条件：

> `KSDA-static < RA - 1%`

严格按实验协议，`D.1+` 本不应作为下一步主线继续推进。

---

## 3. D.1+ 结果：KCAR gate 没有证明优于几何 gate

### 总表

| 方法 | Mean Acc | Std | Δ vs RA |
|------|----------|-----|---------|
| `RA + CSP + LDA` | `0.4375` | `0.1408` | `0.0000` |
| `fixed w=0.5` | `0.3974` | `0.1029` | `-0.0401` |
| `KSDA-geometric-gate` | `0.3897` | `0.1187` | `-0.0478` |
| `KSDA-kcar-gate` | `0.3862` | `0.0952` | `-0.0513` |
| `KSDA-static` | `0.3488` | `0.0650` | `-0.0887` |

数据来源：`results/ksda/exp_d1_plus/2026-03-08-ksda-d1plus/summary.json`

### 直接比较

- `KSDA-kcar-gate` vs `KSDA-geometric-gate`
  - `mean delta = -0.00347`
  - `wins = 3`
  - `losses = 5`
  - `draws = 1`

- `KSDA-kcar-gate` vs `KSDA-static`
  - `mean delta = +0.03742`
  - `wins = 6`
  - `losses = 2`
  - `draws = 1`

- `KSDA-kcar-gate` vs `fixed w=0.5`
  - `mean delta = -0.01119`
  - `wins = 2`
  - `losses = 6`
  - `draws = 1`

### 关键观察

1. **动态 gate 确实能“救回一部分”静态对齐的损失**
   - `KSDA-static` 是 `0.3488`
   - `KSDA-geometric-gate` 升到 `0.3897`
   - `KSDA-kcar-gate` 升到 `0.3862`
   - 说明：在当前 Koopman 管线里，动态控制不是完全没用

2. **但 `KCAR gate` 没有超过 `geometric gate`**
   - 预设标准是：`KCAR` 至少要比 `geometric` 高 `+0.5% ~ +1%`
   - 实际结果是：`KCAR` 还略低 `-0.35%`
   - 因此 **D.1+ 也没有通过**

3. **`fixed w=0.5` 反而最好**
   - 这说明当前 gate 信号并没有把插值强度决策做得比简单固定策略更好
   - 也意味着：
   - **即使进入了 Koopman 空间，当前 gate 逻辑仍然没有稳定转化成更好的动作**

4. **KCAR 机制本身是通的，但信号质量不够**
   - `corr(rho_t, w_t_kcar)` 的平均值约为 `-0.724`
   - 这说明 `ρ↑ → w↓` 这一门控映射按预期工作
   - 所以问题不在门函数本身，而在：
   - **当前 `ρ_t` 还不足以稳定指示“更好的对齐强度”**

---

## 4. 这轮结果最重要的诊断：D.1 和 D.1+ 的耦合度过高

这轮实验最大的价值，不只是“KSDA 没过线”，而是暴露了一个更关键的方法学问题：

> **当前 `D.1+` 对 `D.1` 的静态对齐器 `A` 依赖过强。**

具体表现是：

1. `D.1` 的静态基座已经明显不行  
   `KSDA-static` 比 `Koopman-noalign` 还差很多，说明 `A` 本身就带来负迁移。

2. `D.1+` 的所有 gate 都是在这个偏弱的基座上做“修补”  
   这会使得：
   - `KCAR gate` 的真实能力被 `A` 的错误放大或掩盖
   - `geometric gate` 与 `KCAR gate` 的比较也被共同的弱基座污染

3. 结果上就会出现一种很难解释的局面  
   - `KCAR` 和 `geometric` 都比 `KSDA-static` 好
   - 但二者都明显不够好
   - 你无法分清：
     - 是 gate 信号本身不行
     - 还是 `A` 太差，导致任何 gate 都被拖垮

### 这意味着什么

本轮结果更支持下面这句话，而不是“KSDA 整体失败”：

> **Koopman 表征值得继续，但当前 Phase 1 里把“静态对齐器质量”和“动态 gate 信号质量”绑得太紧了。**

---

## 5. 这轮结果可以支持什么，不能支持什么

### 可以支持的结论

1. **Koopman 表征本身有信息**
   - 因为 `Koopman-noalign` 明显优于传统 `NoAlign`

2. **当前版本的静态对齐器 `A` 不成立**
   - 因为 `KSDA-static` 显著差于 `Koopman-noalign`

3. **KCAR gate 的机制方向是通的**
   - 因为 `ρ` 与 `w` 呈稳定负相关

4. **当前 `KCAR gate` 尚未证明优于几何 gate**
   - 因为它没有达到预设的 `+0.5% ~ +1%` 优势

### 不能支持的结论

1. 不能说 **KSDA 已经成立**
2. 不能说 **KCAR 在 Koopman 空间里已经优于几何 gate**
3. 不能说 **现在值得继续 `D.2 / D.3 / D.4`**
4. 也不能简单说 **“Koopman 路线失败”**
   - 因为失败集中在 `A`，不是集中在 `Koopman-noalign`

---

## 6. 正式决策

### 本轮决策

**不继续 `D.2 / D.3`。**

理由有三条：

1. `D.1` 已经触发停止条件
2. `D.1+` 没有证明 `KCAR gate > geometric gate`
3. 当前 `D.1 / D.1+` 的高耦合使得继续往后堆复杂度，难以归因

### 当前最合理的研究口径

> **这轮不是“KSDA 完全失败”，而是“当前版本的静态对齐器 `A` 失败，且其失败使得 D.1+ 的 gate 比较不够干净”。**

---

## 7. 对下一步的建议

由于你已经明确觉得当前 KSDA 实验计划耦合度过高，这轮结果其实是支持这个判断的。

因此，下一步更合理的方向不是继续把 `D.2 / D.3` 接上，而是优先做**解耦**：

### 方向 A：先只修静态对齐器

目标是回答：

> Koopman 空间里，是否存在一个更保守、更稳定的静态对齐器，至少能接近 `RA`？

例如只尝试：
- 均值对齐
- 对角缩放
- shrinkage 版 affine
- 或不经过 full covariance whitening 的保守版本

### 方向 B：先单独验证 KCAR 信号，不绑定当前 `A`

目标是回答：

> 在不依赖当前弱 `A` 的前提下，`KCAR` 是否真的比几何信号更适合作为风险/门控信号？

这样可以把：
- `A` 的问题
- `gate` 的问题

拆开验证，而不是继续一起推。

---

## 8. 最终判断

如果只用一句话总结本轮：

> **当前最值得保留的是 Koopman 表征本身；当前最该推翻并重做的是 D.1 的静态对齐器，以及由它带来的 D.1 / D.1+ 高耦合实验设计。**
