# Exp-D3：线性性能指标驱动的保守算子修正

**定位**: KSDA v3 的第二层想法验证  
**目的**: 验证线性性能指标是否能稳定地驱动低秩、保守的 Koopman 算子修正

---

## 修正形式

不直接学全自由度 `ΔK_t`，而是：

`K_tilde = K_hat + α_t U V^T`

其中：
- `U, V` 来自 `K_t0 - K_s` 的 top-1 奇异方向
- `α_t = clip(γ * β^T m_t, -0.1, 0.1)`
- `γ ∈ {0.05, 0.1}`

---

## 目标

这里线性性能指标只负责三件事：
- 要不要修
- 修多少
- 沿哪个固定低秩方向修

---

## 比较对象

- `best single expert`
- `D2 selector`
- `D3 corrected selector`

---

## 通过标准

`D3` 通过必须同时满足：

1. 修正后 `transition residual` 下降
2. 同时分类准确率提升
3. 不出现明显数值发散
4. 单被试灾难性下降不超过 `-2%`
