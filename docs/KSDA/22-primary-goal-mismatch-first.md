# KSDA 当前不可偏离的主目标

**日期**: 2026-03-09

---

## 一句话版本

KSDA 这条线的首要目标不是“先做一个更复杂的动态 controller”，而是：

> **先寻找、定义并验证一个“脑电表征—行为不一致性”的度量；再用这个度量去驱动 Koopman 特征空间中的对齐与 Koopman 算子的线性性能修正。**

---

## 正式表述

当前主目标固定为三步：

1. **问题定义 / 度量定义**
   - 寻找一个能够刻画“脑电表征与行为决策不一致”的度量
   - 用它证明：
     - 这种问题确实存在
     - 现有方法也普遍存在这个问题

2. **方法层第一步**
   - 在 **Koopman 特征空间** 中做跨被试对齐
   - 不是回到原始协方差空间里做主方法

3. **方法层第二步**
   - 用 **线性性能指标** 修正 Koopman 算子的估计
   - 让算子更新不再只由无监督动力学残差决定

---

## 明确不是当前主目标的东西

以下内容都不是当前阶段的最终目标，只能算中间手段：

- 单纯追求更复杂的 gate / controller
- 把候选动作选择本身当成主要贡献
- 只比较哪个静态器更强，而不回到 mismatch metric
- 在没有清楚 mismatch 定义前就直接堆闭环策略

---

## 如何判断当前工作有没有偏题

以后每一步实验都要能回答下面两个问题：

1. **这一步是否在帮助我们定义、验证或操作化 mismatch metric？**
2. **这一步是否在帮助我们把 mismatch metric 变成 Koopman 空间中的对齐/算子修正机制？**

如果两个问题都答不上来，就说明当前实验已经偏离主目标。
